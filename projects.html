Personal Projects

MTGADraft

A WebApp allowing Magic the Gathering: Arena players to draft with others players using card they own in the game (determined by parsing the game's logs). Built using Node.js (express and socket.io) on the backend and Vue.js on the frontend.

Vulkan

Little project to learn the Vulkan API and try out some new rendering techniques. Repository.

  • Rasterisation and Raytracing hybrid: Rasterized GBuffer and additional raytraced passes for effects.
  • Diffuse Global Illumination (DDGI)
  • Raytraced Shadows and Reflections

OpenGL Experiments

Learning OpenGL the modern way! I leave here my takes at some rendering (or just OpenGL related) techniques. Examples:

  • Forward Rendering and some post processing ("God Rays").
  • Deferred Rendering and optimizations (Tiled Lightning using a compute shader).
  • Raymarching and raytracing in fragment shader (Combined with other thechniques, or not! Take a look at my ShaderToy profile).
  • Physical animation (simple particles, buoyancy, shallow water (Saint Venant equation)... See this short video)

... among others. This is just meant to be a way for me to learn these techniques.
See here for some code samples: Current engine repository and Some older experiments.
