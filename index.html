Personal Projects

OpenGL Experiments

Learning OpenGL the modern way! I leave here my takes at some rendering (or just OpenGL related) techniques. Examples:

  • Forward Rendering and some post processing ("God Rays").
  • Deferred Rendering and optimizations (Tiled Lightning using a compute shader).
  • Raymarching and raytracing in fragment shader.
  • Physical animation (simple particles, shallow water (Saint Venant equation)...)
... among others. This is just meant to be a way for me to learn these techniques. More informations/Repository

SenOGL

Little library wrapping OpenGL object and functionality. This is intended for personal use and in early stage so I can't recommend its use (also, I'm not satisfied with the current API, it may change significantly). Repository
